# 多功能圖片影音製作器 V5.3

[![License](https://img.shields.io/badge/License-LGPL%20%2F%20GPL-blue)](LICENSE)
[![Python](https://img.shields.io/badge/Python-3.11%2B-green)](https://www.python.org/downloads/)
[![Windows](https://img.shields.io/badge/Platform-Windows-blue)]()

這是一個功能完整的 Windows 桌面工具，可將多張圖片、影片片段、個別音訊、背景音樂、轉場效果與字幕合成為專業級 MP4 影片。

## ✨ 主要特性

- 🎬 **多媒體合成**：支援圖片、影片、音訊等多種格式的混合合成
- 🎨 **豐富轉場效果**：提供 15+ 種 ffmpeg xfade 轉場效果（淡入淡出、擦拭、滑動等）
- 📝 **靈活字幕系統**：全域與逐素材字幕設定，支援多種顏色、位置、大小與背景框
- 🎵 **音訊混音**：支援個別音訊與背景音樂同時混音
- 🎯 **影片剪輯工具**：內建預覽、剪輯、分割功能，支援 ffplay 即時播放
- 📊 **多解析度輸出**：支援 480P 至 4K，可自由選擇 16:9 或 4:3 比例
- 📦 **單檔 EXE 打包**：exe 內含所有依賴（Python、ffmpeg、字型、DLL），開箱即用

本專案已準備好打包為單一 exe：
- ✅ exe 內含 `ffmpeg`、`ffprobe` 與必要 DLL
- ✅ exe 內含完整中文字型 `NotoSansTC-VF.ttf`（支援繁體中文）
- ✅ exe 內含 Windows 應用圖示 `assets/app.ico`
- ✅ exe 啟動與匯出時不顯示 CMD 視窗
- ✅ 實時進度條顯示匯出進度

## 🚀 快速開始

### 對於一般使用者（推薦）

如果只想使用現成的 exe，無需安裝 Python：

1. 下載 [Releases](../../releases) 中的最新 `多功能圖片影音製作器V5.2.exe`
2. 雙擊 exe 直接執行（首次啟動需等待數秒進行解壓縮）

1. Clone 本倉庫
2. 安裝 Python 3.11+ 與依賴：`pip install -r requirements.txt`
3. 執行原始碼：`python 多功能圖片影音製作器V5.py`
4. 修改後重新打包：`pyinstaller 多功能圖片影音製作器V5.spec`


如果看到類似 `Python 3.13.5`，代表安裝成功。

### 2. 安裝套件

進入本專案資料夾：

安裝必要套件：

```powershell

python -m pip install -r requirements.txt
```

`requirements.txt` 內含：

- `pyinstaller`：打包 exe

### 3. 測試 Python 原始碼可執行

## 三、重新打包 exe 教學

本專案已包含打包所需資源：

- `ico.png`：程式視窗圖示
- `assets/app.ico`：exe 檔案圖示
- `ffmpeg/bin/`：ffmpeg、ffprobe 與 DLL
- `fonts/NotoSansTC-VF.ttf`：字幕用中文字型
- `licenses/`：第三方元件授權文字
- `多功能圖片影音製作器V5.spec`：PyInstaller 打包設定
- `build_exe.ps1`：一鍵打包腳本

### 方法 A：使用一鍵打包腳本

在 PowerShell 進入專案資料夾後執行：

```powershell
Set-ExecutionPolicy -Scope Process Bypass
.\build_exe.ps1
```

### 方法 B：手動執行 PyInstaller

```powershell
pyinstaller 多功能圖片影音製作器V5.spec
```

完成後 exe 會在：

```text
dist\多功能圖片影音製作器V5.2.exe
```

打包設定重點：

- `console=False`：不開 CMD 視窗
- `icon=assets/app.ico`：套用 exe 圖示
- `ffmpeg/bin` 被加入 `binaries`：讓 exe 內含 ffmpeg 與所有 DLL
- `fonts/NotoSansTC-VF.ttf` 被加入 `datas`：讓字幕不依賴其他電腦的中文字型

## 四、軟體詳細功能說明

### 1. 專案管理

- **開啟新檔**：清空目前所有圖片、影片、音樂清單與設定，建立全新專案。
- **開啟舊檔**：讀取先前儲存的 `.json` 專案檔，還原素材清單與所有設定。
- **另存新檔**：將目前素材路徑、字幕、音訊、背景音樂、轉場等所有設定儲存成 `.json` 專案檔。

> 注意：專案檔保存的是素材的**路徑**，不會把圖片、影片、音樂複製進專案檔。若搬到其他電腦，素材也要一起搬移並保持路徑可存取。

---

### 2. 匯入圖片與影片

- 支援一次選取多張圖片（JPG、PNG 等常見格式）或多支影片（MP4、MOV、AVI 等）。
- 每個素材以縮圖卡片方式顯示在清單中，並標示順序編號。
- **圖片**：可在卡片上設定停留秒數（預設 3 秒，可調整）。
- **影片**：自動讀取媒體長度，顯示原始時長；可進行裁切剪輯（見下節）。
- 清單中的素材可透過**上移 / 下移**按鈕調整播放順序。
- 可個別刪除清單中的某一項素材。

---

### 3. 影片剪輯（雙擊影片開啟）

雙擊清單中的影片卡片，會開啟「影片剪輯」對話框，提供以下功能：

#### 3-1. 畫面預覽
- 左側顯示 560×315 的畫面預覽區，預設顯示起點畫面。
- 拖曳「預覽位置」滑桿，可即時抽幀顯示對應時間點的靜態畫面（防抖 300ms 後才抽幀，避免過度呼叫 ffmpeg）。

#### 3-2. 播放預覽（ffplay）
- 按「▶ 播放預覽」開啟獨立播放視窗，使用 ffplay 從起點播放至終點。
- 播放視窗帶有可拖曳的時間軸滑桿，可隨時跳至任意時間點。
- 播放視窗支援「⬅ 設為起點」與「設為終點 ➡」按鈕，可直接把目前播放位置回傳給剪輯對話框。
- 播放視窗自動貼合於控制視窗上方（Windows API 同步定位）。

#### 3-3. 裁切區間設定
- 獨立的「起點」和「終點」滑桿，拖曳即可設定裁切範圍。
- 每個滑桿旁有「← 設為預覽位置」按鈕，可快速把目前預覽幀設為起點或終點。

#### 3-4. 分割影片
- 拖曳「分割點」滑桿，設定分割時間點。
- 按「確認分割」後，該影片會在清單中被拆分成兩段（前段 + 後段），各自可獨立設定音訊與字幕。

---

### 4. 個別素材的音訊與字幕

每個圖片或影片卡片可獨立設定：

- **個別音訊**：上傳專屬音訊（MP3、WAV 等），只在該素材播放期間出現。上傳後會顯示音訊檔名，並可按按鈕移除。
- **字幕文字**：手動輸入要顯示在該素材畫面上的字幕文字（若有個別音訊，預設會自動填入音訊檔名）。
- **字幕位置（個別）**：可選擇「下方」或「置中」，覆蓋全域位置設定。
- **字幕大小（個別）**：可單獨調整該素材的字體大小，覆蓋全域大小設定。

> 字幕文字欄位只有在主介面勾選「啟用字幕」時才會顯示。

---

### 5. 背景音樂

- 可匯入一個或多個背景音樂檔（MP3、WAV 等）。
- 每段音樂可設定**開始時間**與**結束時間**（裁切範圍）。
- 可調整每段背景音樂的**音量**（0%～200%）。
- 多段背景音樂會依序串接後，與影片中的個別音訊混音。
- 背景音樂可設定是否循環補足整部影片長度。

---

### 6. 字幕全域設定

在主介面的字幕設定區塊可調整：

- **啟用字幕**：勾選後，所有有字幕文字的素材都會在畫面上顯示字幕。
- **顏色**：白色、黃色、淡綠色、粉紅色。
- **位置**：下方（畫面底部）或置中。
- **字體大小**：10～500，預設 70。
- **字體樣式**：預設為**粗體**（bold），確保字幕在各種背景下清晰可見。
- **字幕背景框**：勾選「背景框」後，字幕後方會顯示半透明色塊，增加可讀性。
  - 背景框顏色：黑色、藍色、深灰色、紅色。
  - 透明度：0%～100%（預設 50%）。

---

### 7. 影片比例與解析度

可從主介面選擇：

- **比例**：16:9（橫向）或 4:3（傳統）。
- **解析度**：
  - 480P（854×480）
  - 720P（1280×720）
  - 1080P（1920×1080）
  - 2K（2560×1440）
  - 4K（3840×2160）

解析度越高，匯出時間越久，輸出檔案也越大。不足解析度的素材會自動縮放並填黑邊，不會拉伸變形。

---

### 8. 轉場效果

- **無轉場**：各素材直接接續，無過渡效果。
- **ffmpeg xfade 轉場**：提供多種轉場，包括：
  - fade（淡入淡出）
  - wipeleft / wiperight / wipeup / wipedown（擦拭）
  - slideleft / slideright / slideup / slidedown（滑動）
  - circlecrop（圓形裁切）
  - rectcrop（矩形裁切）
  - distance（距離）
  - fadeblack / fadewhite（淡入黑/白）
  - radial（放射）
  - smoothleft / smoothright / smoothup / smoothdown（平滑滑動）
  - 以及其他 ffmpeg 支援的 xfade 效果

- **影片素材套用轉場**：勾選此選項後，影片片段之間也會套用轉場（預設僅圖片之間有轉場）。

---

### 9. 匯出影片

> ⚠️ **重要：所有檔案路徑請勿包含中文字**（包括圖片、影片、音訊、背景音樂的路徑，以及 exe / 專案檔所在資料夾）。路徑含中文可能導致 ffmpeg 無法讀取素材或匯出失敗。建議將所有素材與程式放在純英文路徑下，例如 `C:\Videos\project\`。

1. 確認至少已匯入一個圖片或影片素材。
2. 按下「**匯出影片**」按鈕。
3. 匯出期間主介面顯示進度條，不會跳出 CMD 視窗。
4. 匯出分為兩個階段：
   - **第一階段**：將每個素材（含字幕、個別音訊）轉換為統一規格的暫存 MP4 片段。
   - **第二階段**：依轉場設定串接所有片段，並混入背景音樂，輸出最終 MP4。
5. 匯出完成後彈出結果視窗，顯示輸出檔案路徑。
6. 最終輸出檔案：

```text
output_video.mp4（輸出在 exe 所在資料夾）
```

> 若同資料夾已有 `output_video.mp4` 且正被播放器開啟，請先關閉播放器再重新匯出。

---

### 10. 最終影片預覽

- 匯出完成後可按「**播放最終影片**」，使用系統預設媒體播放器開啟 `output_video.mp4`。
- 介面右側提供內建的簡易預覽播放器（QMediaPlayer），可在不離開程式的情況下快速確認輸出結果。

## 五、常見問題

### exe 很大是正常的嗎？

正常。因為單一 exe 內含 Python 執行環境、PyQt5、ffmpeg、ffprobe、DLL 與中文字型，檔案體積較大屬於預期行為。

### 第一次啟動比較慢？

正常。PyInstaller 單檔模式（onefile）啟動時會先將內含資源解壓縮到系統暫存資料夾，需要數秒鐘。

### 防毒軟體提示未知程式？

自行打包的 exe 沒有程式碼簽章時，Windows SmartScreen 或防毒軟體可能提示未知來源。若要公開發佈，建議使用程式碼簽章憑證簽署 exe。
