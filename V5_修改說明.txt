多功能圖片影音製作器 V5.2 修改說明
修改日期：2026-05-11
===================================================================

■ 本次新增功能：影片雙擊剪輯視窗

  【觸發方式】
  在圖片/影片清單中，對任意「影片」項目連點兩下，即可開啟剪輯對話框。

  【V5.2 播放改進：內嵌小視窗 + 停止時設定起點/終點】

  1. 播放內嵌（不再全畫面）
     - 按「▶ 播放預覽（內嵌）」後，ffplay 視窗會直接內嵌在
       對話框左側的預覽區域內播放，不跳出外部全螢幕視窗。
     - （Windows 平台透過 SetParent API 內嵌；其他平台仍在小視窗播放）
     - 播放期間：預覽位置滑桿會自動跟著播放進度移動。
     - 右側顯示目前播放位置（mm:ss.t）。

  2. 停止播放 → 詢問設定起點/終點
     - 按「⏹ 停止播放」後，程式會記錄目前的播放時間點，
       並顯示選擇對話框：
       ┌──────────────────────────────────┐
       │ 播放停止於 xx:xx.x               │
       │                                  │
       │ [⬅ 設為起點] [設為終點 ➡] [不設定] │
       └──────────────────────────────────┘
     - 選擇後，對應的起點/終點滑桿會自動更新。
     - 播放自然播完結束時，也會自動更新預覽畫面到結束位置。

  【其他剪輯功能（與 V5.1 相同）】

  3. 幀預覽
     - 拖曳「預覽位置」滑桿，停止 0.3 秒後自動抓取該秒畫面顯示。

  4. 設定裁切區間（起點 / 終點）
     - 拖曳起點/終點滑桿，或按「← 設為預覽位置」快速設定。
     - 起點/終點互鎖保護，防止起點超過終點。
     - 按「✔ 套用裁切設定」確認，匯出時套用 ffmpeg -ss/-to。

  5. 分割影片
     - 拖曳分割點滑桿後按「✂ 確認分割（分成兩段）」。
     - 原影片從清單移除，兩段新影片自動插入原位置。

  【清單項目顯示】
  - 影片項目顯示 🎬 圖示、裁切區間資訊及「連點兩下可剪輯」提示。

  【匯出 / 存讀檔】
  - trim_start / trim_end 儲存至專案 JSON，開舊檔可還原。
